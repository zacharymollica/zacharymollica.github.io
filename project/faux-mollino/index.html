Faux Mollino Table

2018

Scale reproduction of an oak and glass table designed by famed Italian designer Carlo Mollino.

A pair of hand sketched elevations by the designer served as a starting point for the physical reproduction of this well known piece. A careful digital 3D reconstruction was drawn up from these 2D images before moving on to CNC fabrication. This exercise became the inspiration for an induction studio project which was further developed through cohorts of Design + Make students.
